# Chat App

A modern, cross-platform chat application built with Flutter.

## Features

* **Home Screen**: Lists recent conversations with quick previews of the last message.
* **Chat Screen**: A dedicated room to view conversation history with distinct message bubbles for sent and received texts.
* **Modern UI**: Implements Material 3 design guidelines with support for both light and dark system themes.
* **Responsive Layout**: Designed to look great on mobile and scalable for other platforms.

## Tech Stack

* Flutter / Dart
* Material 3 Design System

## Getting Started

To run this application locally:

1. Ensure you have Flutter installed and set up.
2. Clone this repository.
3. Fetch dependencies: `flutter pub get`
4. Run the app: `flutter run`
